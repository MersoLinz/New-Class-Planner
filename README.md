# ClassPlanner
Die zentrale Plattform für Mitschriften, Schulaufgaben und Termine in der Klasse
# Firebase
Wir verwenden Firebase Firestore als zentrale Datenbank, um Termine, Tests und Inhalte klassenweit zu speichern und für alle Nutzer synchron verfügbar zu machen. Über Firebase können sich Lehrkräfte und Schüler anmelden, wodurch alle Daten sicher gespeichert, gemeinsam genutzt und rollenbasiert verwaltet werden können.
